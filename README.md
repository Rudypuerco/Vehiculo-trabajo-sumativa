# Vehiculo-trabajo-sumativa

Requerimientos Funcionales 

-El sistema permite ingresar los siguientes datos: marca, el modelo,
	la cilindrada, tipo de combustible, capacidad de pasajeros

-El sistema proyecta los datos ingresados por el usuario


Requerimientos no funcionales
- El sistema se ejecuta por consola 
- Código documentado
- Compatible con diferentes sistemas operativos
- Fácil de usar


Usuario 1

Necesita ingresar datos/detalles de un vehículo para
visualizar y ordenarse de mejor forma 

Criterios de aceptación 

debe poder ingresar los datos que se solicitan
debe mostrar datos de forma correcta 
no puede presentar errores al ingresar datos lógicos


Usuario 2

El usuario necesita ver los datos en la pantalla 
para confirmar que se ingresaron correctamente los datos

criterios de aceptación

los datos deben ser los mismos que ingreso el usuario
el sistema debe mostrar mensajes claros
